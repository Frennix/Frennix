### Grant founder role (first time only)

After migration, promote your account:

```sql
INSERT INTO staff_memberships (user_id, role)
VALUES ('YOUR-USER-UUID', 'founder')
ON CONFLICT (user_id) DO UPDATE SET role = 'founder', revoked_at = NULL;
```

3. Open app → Settings → **Founder Dashboard**
4. Verify sidebar navigation (desktop) or hamburger menu (iPhone)

6. Confirm non-staff users see "Staff access required"
