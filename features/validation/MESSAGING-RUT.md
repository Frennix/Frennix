# Messaging — Beta tester runbook

**Time:** ~10 minutes

## Steps

1. Open **Messages** tab — confirm conversation list loads.
2. Open a chat from a training match or trainer connection.
3. Send a text message — confirm realtime delivery on other device.
4. Background the app — send a message from the other user — confirm push notification.
5. Tap push — confirm opens correct chat (cold start if possible).
6. Confirm unread badge clears after opening chat.

## Report issues

**Messages** tab footer → **Report an issue**, or Settings → Send feedback (area: Messages).
