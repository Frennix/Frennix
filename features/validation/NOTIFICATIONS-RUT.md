# Notifications — Beta tester runbook

**Time:** ~10 minutes

## Steps

1. Open **Settings → Notification settings** — enable push if prompted.
2. Trigger a **Training Match** notification — confirm copy says "Training Match" / partner message wording.
3. Trigger a **trainer connection request** (trainer account) — confirm in-app notification.
4. Trigger a **coaching request accepted** (athlete account).
5. Tap each notification — confirm lands on chat or connections screen.
6. Disable **Training matches** toggle — confirm match push suppressed on next test (if feasible).

## Report issues

Settings → Send feedback (area: Notifications).
