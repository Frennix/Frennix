# Events — Beta tester runbook

**Time:** ~10 minutes

## Steps

1. Open **Events** tab — list loads.
2. Tap **+ Create** — create a simple workout event with title, time, location.
3. From a second account, open the event and **Join**.
4. Confirm host receives notification (if applicable).
5. Leave event — confirm attendee count updates.

## Report issues

**Events** tab footer → **Report an issue**, or Settings → Send feedback (area: Events).
